# The Rise of the Dark Prince

> *A dark fantasy tale of ambition, betrayal, forbidden power, and the rise of a prince who will reshape the world... or burn it down.*

**Welcome to the official Story Bible repository.**

This living document collection ensures consistency across the novel, webnovel serialization, and future webtoon adaptation. Everything you need to build a rich, coherent world is here.

## How to Contribute & Use
- Edit any Markdown file directly in GitHub or your editor
- Commit changes with clear messages (e.g. `docs: update Dark Prince character arc`)
- The Story Bible is a **living document** — update it as the story evolves
- For major changes, open a Pull Request
